2021-03-11 题解

题意：给定nnn，xxx，yyy，初始值为零，每次加一或减一消耗xxx，乘二消耗yyy，求造出值nnn的最小消耗。

减一导致了后效性，因此本题的主要思路是消除后效性。

  • 当iii为奇数时，f(i)f(i)f(i)由f(i−1)f(i-1)f(i−1)和f(i+1)f(i+1)f(i+1)转移来，由偶数结论可知，f(i+1)f(i+1)f(i+1)由f((i+1)/2)f((i+1)/2)f((i+1)/2)得出，因此f(i)=min(f(i−1)+x,f((i+1)/2)+x+y)f(i)=min(f(i-1)+x,f((i+1)/2)+x+y)f(i)=min(f(i−1)+x,f((i+1)/2)+x+y)；
#include<bits/stdc++.h>
using namespace std;
#define int long long
#define Min(x,y) (x<y?x:y)
int n,x,y,f[10000010];
signed main() {
    scanf("%lld%lld%lld",&n,&x,&y);
    f[1]=x;
    for (int i=2;i<=n;i++) {
        if (i%2==0) f[i]=Min(f[i-1]+x,f[i/2]+y);
        else f[i]=Min(f[i-1]+x,f[(i+1)/2]+x+y);
    }
    printf("%lld\n",f[n]);
    return 0;
}
